# MDBT42Q-breakout

Raytac社製nRF52832モジュールMDBT42Qのブレークアウト基板です。

RedBearLabのBLE Nanoの書き込み機(Daplink)を直接挿して書き込みできるようにしました。

部品はコンデンサ、インダクタはすべて1608サイズ、水晶振動子はFC-135です。
C2,C3の容量は水晶振動子の負荷容量に合わせて選定してください。
